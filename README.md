# Free NHL pool

[slapshot.xyz](https://slapshot.xyz)

built with next-js shadcn and rust, [See backend repository](https://github.com/jcorriveau23/backend-pool-nhl)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Run the linter:

```bash
npm run lint
```

Build the ui server:

```bash
npm run build
```
